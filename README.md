使用方法

	点击download zip下载最新源码，修改文件名为CodeSimple，解压上传

	后台配置外观选项，包括网站LOGO、网站favicon、网站banner图片、首页顶部文字

、首页顶部文字 h3等等，友情链接6个位置，不用插件就可以。

 注意事项

	文件夹名必须CodeSimple，修改无效。
